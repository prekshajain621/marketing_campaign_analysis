# marketing_campaign_analysis
## Description
The analysis is provided for designing and development of marketing campaign Performance dashboard using Excel, SQL and PowerBI. The purpose of creating this dashboard is to gain insightful insights to maximize the profit for campaign Performance.
Tools Used:- Excel, SQL, Power BI 
📁 Dataset:- More than 2000 Customer profile from maven analytics.
